# Sprint 2.3 — Public Web Application

## Delivered
- Mobile-first Stewart Park Intelligence Center
- Searchable Council records
- Expandable five-year timeline
- Phase I and Phase II feature sections
- Funding-event display
- Verified Questions with Evidence Explorer
- Official source library
- Verification-status disclosure
- Static deployment model for GitHub Pages

## Preview

Open `website/index.html` directly, or run:

```bash
python scripts/serve_web.py
```

## Current limitation

Motions, votes, minutes links, and video timestamps remain pending direct verification and therefore are not presented as complete.
