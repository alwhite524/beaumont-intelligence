# Import and Verification Workflow

## Agenda items
1. Create or locate the meeting.
2. Add the agenda item with the exact item number and official title.
3. Link the agenda, staff report, attachments, and minutes.
4. Keep the timestamp status `pending` until the official player is reviewed.

## Council actions
1. Add one action record for each distinct motion or authorization.
2. Preserve exact motion and vote language when available.
3. Link the minutes or staff report page used as evidence.
4. Mark `verified` only after direct source review.

## Funding events
Record each award, application, allocation, amendment, pause, clearance, or funding-status statement separately. Do not overwrite earlier amounts; preserve the date and status of each amount.

## Project features
Features are controlled project components, not free-form tags. Link them to documents, agenda items, and media so the archive can answer when a feature first appeared, was rendered, approved, and completed.
