# Beaumont Intelligence Document Intelligence

## Purpose

The Document Intelligence subsystem provides a browser-based interface for exploring official City of Beaumont public records.

Rather than downloading PDFs directly, residents can:

- Read documents in the browser
- Navigate related attachments
- Return to the associated Council Intelligence briefing
- Explore related Intelligence Centers
- Search official records

---

## Components

index.html
Official Source Library

viewer.html
Universal document viewer

document-data.js
Metadata describing every official document
